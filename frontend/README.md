# AI Q&A Frontend

Single-page frontend for the AI Q&A application.

Built with HTML, CSS, and JavaScript. Connects to the Django REST API backend deployed on Hugging Face Spaces.

## Features

- Upload PDF, audio, and video files
- Ask questions against uploaded documents
- Real-time AI responses powered by Google Gemini
- Clean, responsive interface

## Backend

API hosted at: https://gsuhani17-ai-qa-backend.hf.space
